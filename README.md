1. Conductivity to Salinity Conversion
2. Salinity to Conductivity Conversion
3. Specific Volume Anomaly and Density Anomaly of Seawater
4. Pressure to Depth Conversion
5. Freezing Point Temperature of Seawater
6. Specific Heat of Seawater
7. Adiabatic Lapse Rate
8. Potential Temperature
9. Sound Speed in Seawater

## How to run
1. Download `UAS11.exe`
2. Run from Command Prompt or double-click
3. Follow on-screen input instructions

## Compiler
Compiled using `gfortran` on Windows.
